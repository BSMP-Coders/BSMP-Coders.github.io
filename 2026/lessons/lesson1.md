## Goal

By the end of this class, you will build a web app from scratch — multiple pages, an AI chatbot, and a real **assistant** with a custom persona, conversation memory, and streaming replies like ChatGPT — all using Copilot prompts.

---

## How This Works

You start with **nothing**. You paste prompts into Copilot, review what it generates, and keep building. Five stages, one continuous build:

| Stage | What You Build | New Skills |
|-------|---------------|------------|
| 1. HTML | A styled webpage with an image gallery | HTML, CSS, linking pages |
| 2. Flask | A Python web server that serves your pages | Routes, templates, server vs client |
| 3. AI Chat | A chatbot powered by Azure OpenAI | API keys, environment variables, fetch |
| 4. Assistant | Personas + conversation memory | System prompts, message history |
| 5. Streaming | Token-by-token replies like ChatGPT | Streaming responses, readable streams |

Each stage builds directly on the one before it.
