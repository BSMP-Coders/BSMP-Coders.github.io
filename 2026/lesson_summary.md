# BSMP 2026 Coding & AI Journey 🌐🚀 <!-- {docsify-ignore-all} -->

Welcome to the **BSMP 2026 Coding & AI Build Academy!**

This course is part of Microsoft's BAM Summer Mentorship Program and is designed to help high school students build confidence and creativity through real-world coding and AI experiences.

This year, **everyone follows one shared curriculum** built around six hands-on lessons. The **Intermediate** and **Advanced** tracks cover the same core concepts and build the same kinds of apps — the **Advanced track** just moves at a faster pace and adds extra challenges on top of each lesson.

Throughout this program, you'll **Learn** the fundamentals of programming with Python, how to use GitHub Copilot in agent mode, and how modern AI works; **Build** real web apps, AI chatbots, agents, dashboards, and games; and **Create** projects that tell stories, solve problems, and reflect your ideas.

You'll gain hands-on experience with GitHub, GitHub Codespaces, AI literacy, and Python-based tools while preparing to showcase your final project to your peers and mentors.

## 🔑 Getting Set Up

Before Lesson 1, make sure you can:

- ✅ Sign in to **GitHub** (use your personal email, **not** your BAM account)
- ✅ Open the lesson repository in **GitHub Codespaces** (one click — no installs needed)
- ✅ Add the class **AI model** to GitHub Copilot (your instructor will share the endpoint, key, and setup steps)
- ✅ Run a lesson in **Copilot agent mode** and complete your first prompt

> 📘 Need help getting started? See [Getting Started with GitHub and GitHub Copilot](/students/github_copilot_setup.md)
