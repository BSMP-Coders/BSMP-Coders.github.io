# Welcome to the BSMP26 Coding & AI Build Academy 👋 <!-- {docsify-ignore-all} -->

Welcome to the **BSMP26 Coding & AI Build Academy**, part of the 2026 **BAM Summer Mentorship Program (BSMP)** powered by Microsoft. This exciting, hands-on program invites high school students to explore the world of programming and artificial intelligence through our **Learn, Build, Create** journey.

This year is bigger and bolder. With AI now at the center of how software gets built, you'll spend less time wrestling with syntax and more time **designing, building, and shipping real AI-powered apps** — using the same tools professional developers use every day. Whether you're just starting your coding path or you're ready to build your own AI agents, this academy is designed to unlock your creativity and technical potential. Led by dedicated Microsoft mentors, instructors, and on-site support staff, you'll gain skills that go far beyond the summer.

---

## Courses Offered 📚

This year we run **one shared curriculum with two tracks**, so everyone learns the same core concepts and builds the same kinds of apps — the **Advanced track** simply moves faster and layers on extra challenges. 

* 👉 **Intermediate Track** - Use Python, GitHub Copilot (agent mode), and AI tools to build web apps, AI chatbots, data dashboards, and games — no experience needed!
* 👉 **Advanced Track** - Go deeper with AI agents, MCP servers, RAG, voice and image generation, and creative coding. Ideal for students with Python and web dev experience.

## Program Themes 🎯

* **Learn**: Grasp the fundamentals of Python, AI, and web technologies in a collaborative, student-friendly environment.
* **Build**: Use GitHub Copilot, Visual Studio Code, and modern frameworks to build real web apps, AI agents, and intelligent tools.
* **Create**: Design and present original coding projects that solve real problems or bring your imagination to life.

---

## Key Components 🔧

* **GitHub Copilot (Agent Mode)**: Your AI pair-programmer. This year you'll work hands-on in *agent mode* — describing what you want and collaborating with Copilot to build it.
* **GitHub Codespaces**: A full coding environment in your browser — one click and you're ready to build, no installs required.
* **VS Code**: The professional-grade editor we'll use throughout the course (right inside Codespaces).
* **Python**: Our core programming language — versatile, powerful, and beginner-friendly.
* **Web App Development**: Build interactive web apps and dashboards with Python (Flask), HTML, and Chart.js.
* **AI Tools & Models**: Work with Azure OpenAI models, build AI chatbots, **agents, custom tools, and MCP servers**, and experiment with RAG, text-to-speech, and image generation.

---

## Learning Outcomes 🎓

By the end of this program, you will:

* Gain hands-on experience in Python programming and web development.
* Build AI-powered apps using GitHub Copilot in agent mode.
* Understand modern AI concepts — chatbots, agents, tools, MCP, and RAG — and how to apply them.
* Explore responsible and ethical AI, and why it matters in what you build.
* Build and share your own projects using GitHub and Microsoft technologies.
* Present your work to mentors and peers in a final showcase video.
* Walk away with a GitHub portfolio that shows off your new skills.
